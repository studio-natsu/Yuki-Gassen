## ❄️ ゲーム概要 / Features
- 🕹️ 自キャラを操作して雪玉を避けろ！
- ⛄ Pyxelならではのドット感＆レトロBGM

| 操作       | キー         |
| -------- | ---------- |
| ゲームスタート | クリック        |
| 移動       | ← → ↑ ↓ キー |\
| ゲーム再スタート | クリック        |


## 🧪 インストール方法 / How to Run

1. Python 3.xをインストール  
2. 以下のコマンドでPyxelを導入：

```bash
    pip install pyxel

```

### 🧊 使用ライブラリ
